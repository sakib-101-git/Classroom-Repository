# Classroom Repository Website

A simple and efficient Classroom Repository Web Application built with HTML, CSS, JavaScript, and Node.js.
This project allows teachers and students to upload, download, and manage classroom materials through a clean and responsive interface.

#🚀 Features

🗂️ File Upload System – Upload notes, assignments, PDFs, images, or any classroom-related material.

📁 Organized Repository – Files are grouped neatly for easy browsing.

👨‍🏫 Teacher & Student Friendly – Simple interface designed for ease of use and locate the necessary notes

🌐 Node.js Backend – Handles file storage and routing.

💾 Secure and efficient file upload support.

⚡ Frontend – Fully responsive HTML, CSS, and JavaScript UI.

🔒 Secure Access Structure – Proper routing and file-system protection.

#🛠️ Tech Stack 

Frontend:

HTML5

CSS3

JavaScript (Vanilla)

Backend:

Node.js
